# Spring-Data

@LastModifiedDate

```java
    /**
     * 修改时间
     */
    @Column(name = "last_modify_time")
    @LastModifiedDate
    private Date lastModifyTime;
```

@CreatedDate

```java
    /**
     * 创建时间
     */
    @Column(name = "create_time")
    @CreatedDate
    private Date createTime;
```
