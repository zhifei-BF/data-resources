# git命令

下载项目：

```
git clone 项目地址
```

查看本地分支：

```
git branch
```

查看远程全部分支：

```
git branch -a 
```

切换分支：

```
git checkout 分支名
```

合并某个分支：

```
git merge 分支名
```

推送代码：

```
git push 远程分支名
```

删除分支：

```
git branch -D 分支名
```
