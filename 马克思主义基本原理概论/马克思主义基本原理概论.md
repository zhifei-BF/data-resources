## 第一节：马克思主义的产生和发展

### 考点1：马克思主义的产生和发展

1. 马克思主义哲学是时代的产物：

   ①客观条件：资本主义的发展及其内部矛盾尖锐化。

   ②阶级基础：工人阶级作为一只独立的政治力量登上历史舞台，进行反对资本主义制度和资产阶级统治斗争。

### 考点2：马克思主义的直接理论来源

①德国古典哲学，最高成果是黑格尔的辩证法和费尔巴哈的唯物主义，马克思、恩格斯吸取了黑格尔辩证法中的合理思想，彻底批判了它的唯心主义和神秘主义，对它进行了根本改造，创立了唯物辩证法。

②英国古典政治经济学，创始人是威廉配第，其余还有亚当斯密和大卫李嘉图。马克思和恩格斯进一步发展和完善了英国古典政治经济学派的劳动价值论，在此基础上创立了剩余价值学说。

③英、法两国的空想社会主义，代表人物主要有圣西门（法国）、傅里叶（法国）和欧文（英国）。

## 第二节：马克思主义科学性和革命性的统一

### 考点1：马克思主义的科学内涵

①马克思主义是由马克思、恩格斯共同创立的。

②马克思主义不仅包括它的创始人马、恩的理论，也包括它的继承人的理论。特别是列宁、毛泽东、邓小平等人的理论。

③列入马克思主义范畴的理论学说，必须是在基本观点、基本立场、基本方法、价值取向等方面与马克思、恩格斯创立的理论前后一脉相承。在本质上相一致。

④马克思主义理论体系的三个主要组成部分是马克思主义哲学、马克思主义政治经济学、科学社会主义。马克思主义的各个组成部分，一个具有内在逻辑联系的科学体系，其中马克思主义哲学是科学的世界观和方法论，政治经济学是马克思主义的理论基础，处于核心地位的则是科学社会主义理论。

### 考点2：马克思主义的理论特征

马克思主义的理论特征：是以实践为基础的科学性和革命性的统一。

①马克思主义的革命性，集中体现为它的彻底的批判精神，还表现在它具有鲜明的政治立场。

②马克思主义的科学性，首先揭示自然界和人类社会发展的客观规律。其次，马克思主义的科学性还在于它的深刻性。再次，它经受了实践的检验并随着实践的发展而不断发展。

关系：它在实践基础上实现了科学性和革命性的统一，二者是紧紧联系的科学性是革命性的必要前提和保证，革命性是科学性的必然结论和归宿。

### 考点3：马克思主义的理论品质

马克思主义的理论品质：与时俱进

与时俱进的含义。只有体现时代性、把握规律性、富于创造性才是与时俱进。

### 考点4：马克思主义的社会理想

马克思主义的社会理想：社会理想是全部理想的核心

马克思主义最高的社会理想，就是推翻资本主义、实现共产主义，实现共产主义是长期的远大的理想；建设中国特色社会主义是全国各族人民的共同理想，相对于最高理想来说，共同理想是近期的具体理想。

二者关系：最高理想与共同理想，既相互区别又相互联系，二者是辩证统一的关系。首先，共同理想是实现最高理想的必要前提和必要基础。其次，实现共同理想，必须坚持以最高理想为根本方向。

## 第三节：学习、运用和发展马克思主义

### 考点1：学习马克思主义的根本方法

学习马克思主义的根本方法：理论联系实际

把马克思主义作为行动的指南，就是要用马克思主义的立场、观点和方法研究解决问题，必须从客观存在的实际情况出发，而不能从马克思主义的一般原理出发。（如果从一般原理出发，就犯了教条主义错误）

原因：

首先，这是由于马克思主义的实践性决定的。

其次，这是由我们学习马克思主义的目的决定的。

（1）所谓理论联系实际，就是把马克思主义的基本原理作为指导，去观察和分析社会实际、工作实际和思想实际。我国今天所面临的最大的社会实际，就是我国正处于并将长期处于社会主义初级阶段。

（2）要做到理论联系实际，在实际工作中必须反对经验主义和教条主义两种倾向。

## 第一节：物质世界和实践

一、物质世界的客观存在

1. 哲学

   （1）世界观-对整个世界的根本观点；

   （2）哲学-系统化、理论化的世界观。

2. 哲学的基本问题：思维和存在的关系

   （1）第一方面：物质和意识哪个是本原

   |          | 物质         | 意识         |
   | -------- | ------------ | ------------ |
   | 唯物主义 | 本质、第一性 | 派生、第二性 |
   | 唯心主义 | 派生、第二性 | 本质、第一性 |

   （2）第二方面：思维和存在的同一性

   |          | 同一性 |
   | -------- | ------ |
   | 可知论   | 承认   |
   | 不可知论 | 否认   |

3. 关于世界的本质问题

   （1）一元论和二元论

   |        | 世界的统一性 | 本质 |
   | ------ | ------------ | ---- |
   | 一元论 | 承认         | 一个 |
   | 二元论 | 否认         | 二个 |

   （2）唯物主义和唯心主义

   |          | 类型         | 世界的本原   | 举例                      |
   | -------- | ------------ | ------------ | ------------------------- |
   | 唯心主义 | 主观唯心     | 人的主观精神 | 宇宙便是吾心 吾心便是宇宙 |
   | 唯心主义 | 客观唯心     | 某种客观精神 | 道生万物 我命在天         |
   | 唯物主义 | 古代朴素     | 具体物质形态 | 五行说                    |
   | 唯物主义 | 近代形而上学 | 原子         | -                         |
   | 唯物主义 | 辩证唯物主义 | 物质         | -                         |

4. 物质

   （1）定义：不依赖于意识又能为意识所反映的客观存在；

   （2）唯一特性：客观存在性；

   （3）注意：

   ①物质不依赖于意识；（区别：唯心主义）

   ②物质可以被人们认识；（区别：不可知论）

   ③客观实在性是一切物质的共性。（区别旧唯物主义）

5. 物质的根本属性和存在方式：运动

   （1）运动是物质的固有属性：物质和运动不可分；

   （2）运动和静止的关系：绝对和相对的关系。

6. 物质运动的存在方式：时间和空间

   （1）时间：持续性、顺序性；

   （2）空间：广延性、伸张性；

   （3）时间和空间的关系：绝对和相对、无限和有限的统一。

二、实践

1. 定义：主体能动地改造和探索客体的客观物质活动。

2. 构成：

   （1）主体：处于一定社会关系中具有实践能力的人；

   （2）客体：实践活动指向的对象，具有客观性、对象性和社会历史性；

3. 特点：

   （1）客观性：客观的感性物质活动；

   （2）自觉能动性：有意识有目的的活动

   （3）社会历史性：在一定的社会关系中进行的活动。

4. 基本形式：

   （1）物质生产实践；

   （2）处理社会关系的实践；

   （3）科学实验。

5. 人的实践活动与物质世界的关系

   （1）人的实践活动与自然界的二重化。

   （2）社会历史是人们的实践活动创造的。

   （3）实践是人的存在方式。

   实践的观点是马克思主义哲学的基本观点。

三、社会生活在本质上是实践的

1. 劳动实践是人类社会产生的决定性环节；

2. 物质生产实践是人类社会存在的基础；

3. 实践活动是推动人类社会发展的动力。

   实践是人类社会产生、存在和发展的基础，是社会生活的本质。

## 第二节：物质世界的普遍联系和永恒发展

一、联系与发展的普遍性与多样性

唯物辩证法的总特征：联系的观点 和 发展的观点

1. 联系的观点

   （1）客观性：不以人的主观意志为转移；

   （2）普遍性：事物之间；事物内部要素之间；整个世界；

   （3）多样性：直接与间接；内部与外部；本质与非本质；必然与偶然。

2. 发展的观点

   （1）新事物：符合发展规律、具有强大生命力、远大前途；

   （2）旧事物：丧失了存在的必然性、日趋灭亡；

   （3）新事物必然战胜旧事物：

   ​	①符合事物发展的必然趋势；

   ​	②优越于旧事物；

   ​	③符合人民群众的利益。

   （4）两种对立的发展观

   | 唯物辩证法的发展观 | 形而上学的发展观 |
   | ------------------ | ---------------- |
   | 联系的观点         | 孤立的观点       |
   | 发展变化的观点     | 静止不变的观点   |
   | 承认事物内部矛盾   | 否认事物内部矛盾 |

   焦点：是否承认矛盾是事物发展的动力。

二、世界联系与发展的基本规律

1. 对立统一规律（矛盾规律）

   （1）地位：辩证法的实质和核心，根本规律。

   ​	①揭示了事物联系的内容和发展的动力；

   ​	②贯穿其他规律的中心线索和钥匙；

   ​	③最根本的认识方法；

   ​	④唯物辩证法和形而上学的斗争焦点和根本分歧。

   （2）矛盾的同一性和斗争性

   ​	①同一性：相互依存、相互贯通

   ​	②斗争性：相互排斥、相互对立

   ​	③方法论：在对立中把握统一，在统一中把握对立。

   （3）事物发展的根本原因：事物的内部矛盾（内因）

   ​	①内因是事物发展的依据；

   ​	②外因是事物发展的条件；

   ​	③外因通过内因起作用。

   ​	④现实意义：我国的社会主义现代化建设，必须首先依靠本国人民，独立自主，自力更生。

   （4）矛盾的普遍性和特殊性：具体问题具体分析（活的灵魂）

   ​	①相互联结；

   ​	②相互区别；

   ​	③在一定条件下相互转化；

   ​	④现实意义：把马克思主义普遍真理同我国的实际结合起来，建设有中国特色的社会主义。

   （5）矛盾发展的不平衡性-主要矛盾和次要矛盾

   ​	①相互影响：主要矛盾起决定作用，次要矛盾起制约作用；

   ​	②在一定条件下相互转化。

   ​	③现实意义：首先抓住主要矛盾，同时不忽略次要矛盾，注意两者的转化，不失时机地转移工作重点。

   （5）矛盾发展的不平衡性-矛盾的主要方面和次要方面

   ​	①相互制约：主要方面支配次要方面，次要方面影响主要方面；

   ​	②在一定条件下相互转化。

   ​	③现实意义：分清主流和支流，抓住主流，正确认识事物的性质，不忽视支流，注意主流和支流的相互转化。

   （5）矛盾发展的不平衡性-两点论和重点论

   ​	①两点论：在认识事物时，既要看到主要矛盾，又不忽略次要矛盾；在认识某一矛盾时，既要看到主要方面，又要看到次要方面。

   ​	②重点论：认识事物时，抓住主要矛盾；研究某一矛盾时，把握矛盾的主要方面。

   ​	③看问题、办事情既要全面、又要善于抓住重点。

2. 量变和质变及其辩证关系

   （1）含义

   ​	①量变：数量的增减、场所的变更及排列组合的变化；

   ​	②质变：事物根本性质的变化；

   ​	③度：保持质的数量限度。

   （2）量变和质变的辩证关系

   ​	①量变是质变的必要准备；

   ​	②质变是量变的必然结果；

   ​	③量变和质变相互渗透；

   ​	④现实意义：社会主义建设要把远大理想和稳步前进结合起来，反对急躁冒进、急于求成。

3. 否定之否定规律

   （1）唯物辩证法的否定观

   ​	①事物的自我否定；

   ​	②联系和发展的环节；

   ​	③扬弃；

   ​	④现实意义：对一切事物采取分析的态度，反对不加分析的肯定一切或否定一切。

   （2）事物的发展是前进性与曲折性的统一

   ​	①事物的发展经过两次辩证的否定；

   ​	②由肯定到否定，再到否定之否定，表现为螺旋式上升或波浪式前进的过程；

   ​	③事物的发展是前进性和曲折性的统一；

   ​	④两种错误观点：循环论和直线论；

   ​	⑤现实意义：社会主义的前途是光明的，但要准备走曲折的路，

三、世界联系和发展的基本环节

1. 原因和结果：引起和被引起

   （1）只有找出原因，才能认识其本质，提出解决方法；

   （2）正确把握因果联系，才能总结经验教训，指导行动；

2. 必然性和偶然性：确定不移和不确定

   （1）偶然性受必然性支配，必然性决定事物发展的方向和前途；

   （2）必然性通过大量偶然性表现出来，偶然性是必然性的表现形式；

   （3）实际工作中不能忽视偶然性。

3. 可能性和现实性：潜在的和客观存在的

   （1）可能性不等于现实性，一切工作要从现实出发；

   （2）在尊重客观规律的基础上自觉地发挥主观能动性；

   （3）创造有利条件，改变不利条件，争取使好的可能性变成现实从坏处着想，向好处努力。

4. 内容和形式：事物一切要素的总和和表现内容的方式

   （1）根据内容的需要利用旧形式创造新形式；

   （2）首先重视内容，反对忽视内容的形式主义。

5. 本质和现象：根本性质和表面特征

   （1）真相：现象和本质一致；

   ​	  假象：现象和本质不一致；

   （2）认识事物不能只看表面现象；

   （3）对现象要进行深入的分析。
